# Git 常用命令

## 克隆远程仓库
```bash
git clone https://github.com/xxx/xxx.git
```

## 查看远程仓库地址
```bash
git remote -v
```

## 修改远程仓库地址
```bash
git remote set-url origin https://github.com/xxx/xxx.git
```

## 添加远程仓库
```bash
git remote add origin Administrator@server:liuchen/document.git
```

## 删除远程仓库
```bash
git remote rm origin
```
